# Trabalho-3-TRI

Relatório

Para coleta de dados foi utilizado um sistema no javascript chamado localStorage, já que não foi possível realizar o firebase a tempo.
Esse sistema coleta os dados na página de cadastro a partir de getElementById no javascript. Após isso o nome, senha e email registrado poderão ser visualizados na página de perfil, a foto infelizmente não pode ser alterada, mas o nome pode A conta também pode ser deletada quando o usuário quiser. 

Para a coleta de dados é usada a variável const (exemplo: const email = document.getElementById('email').value;, coleta o email do usuário)
